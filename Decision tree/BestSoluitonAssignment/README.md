
# End-to-End Machine Learning Production Project

## Overview

This project demonstrates a complete, production-ready ML workflow:

- Business understanding
- Data preprocessing
- Feature engineering
- Model training & tuning
- Evaluation
- Deployment via FastAPI
- Monitoring & retraining strategy

## How to Run

1. Install dependencies: `pip install -r requirements.txt`
2. Train model: `python src/train.py`
3. Start API: `uvicorn api.app:app --reload`
